== Installation ==
Manual installation is easy and takes fewer than one minute.

1. Download the plugin from WordPress.org, unpack it and upload the [NinjaTeam Facebook Live Chat] folder to your wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to your main WordPress menu > Facebook Live Chat > Settings, and configure the basic options (fan page URL, color, language, where to display, etc)

You're done. Enjoy.
